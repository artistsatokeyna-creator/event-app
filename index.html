📋 初回設定手順（クリックで表示）
① Firebase プロジェクト作成
console.firebase.google.com を開く
→「プロジェクトを作成」→ 名前を入力（例：event-app）→ 作成

② Realtime Database を有効化
左メニュー「構築」→「Realtime Database」→「データベースを作成」
→「テストモードで開始」→「有効にする」

③ Database URL をコピー
データベース画面上部に表示される
https://xxxx-default-rtdb.firebaseio.com/
これを上の「Firebase Database URL」欄に貼り付ける

④ ルームコードを決める
チームで共通の好きな文字列を入力（例：yamada2025）
全員同じルームコードを使うことでデータが共有されます
